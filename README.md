# 🏋️ FitHub CRM – Fitness Center Management System

## 📌 Project Overview

FitHub CRM is a cloud-based Customer Relationship Management (CRM) system developed using Salesforce to manage fitness center operations efficiently. The system centralizes member data, automates class bookings, and provides real-time insights through reports and dashboards.

This project demonstrates how a real-world business problem can be solved using Salesforce CRM by implementing automation, data management, and user-friendly interfaces.

---

* Automate fitness center operations
* Reduce manual work and errors

* Provide real-time reporting and analytics
* Build a scalable and secure CRM system

## 🧩 System Modules

### 1. Member Management

Stores and manages member details such as name, contact, and membership status.

### 2. Trainer Management

Maintains trainer information, specialization, and availability.

### 3. Class Management

Handles scheduling of fitness classes and trainer assignment.

### 4. Booking System

Allows members to book classes and track booking status.

### 5. Automation

Triggers email notifications when a booking is created.

### 6. Reports & Dashboard

Provides insights through reports and visual dashboards.

---

## 🏗️ System Architecture

```plaintext
Users (Admin / Trainer / Member)
        ↓
Salesforce App (FitHub CRM)
        ↓
Custom Objects (Member, Trainer, Class, Booking)
        ↓
Automation (Flow – Email Notifications)
        ↓
Salesforce Database (Cloud)
        ↓
Reports & Dashboards
```

---

## ⚙️ How It Works

1. Admin creates Member and Class records
2. Member books a class
3. Booking record is created
4. Flow triggers email notification
5. Data is stored in Salesforce
6. Reports and dashboards show insights
